# Prompts IA – Scaffold AI CLI

## Prompt 1 – Génération de README

Objectif :
Générer une documentation professionnelle à partir de la structure d’un projet.

Prompt :
Tu es un expert en documentation logicielle.
Génère un README.md professionnel pour un projet nommé "{project_name}".

La documentation doit inclure :

- une description du projet
- la structure des fichiers
- les instructions d’installation
- les instructions d’utilisation
- les technologies utilisées.
